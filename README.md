# Institutional Trading Engine

## Projektname
Institutional Trading Engine

---

## Zweck des Scanners
Dieser Scanner analysiert US-Aktien, ETFs, Indizes und Edelmetalle anhand eines regelbasierten institutionellen Frameworks.

Ziele:
- Marktregime erkennen
- Führende Sektoren identifizieren
- Hochqualitative Swing-Setups finden
- Schwache Assets vermeiden
- Risiko- und Event-Warnungen liefern
- Systematische Watchlists generieren

Der Scanner ist vollständig datengetrieben und verwendet Live-Marktdaten über Polygon.io.
