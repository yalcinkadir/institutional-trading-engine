# PO6 Daily Observation Record Artifact Contract

Status date: 2026-06-01
Status: Done / CI-green

## Purpose

PO6 defines the canonical artifact path contract for PO3 Daily Observation Run Records generated by PO5.

The goal is to keep daily observation records separated from public report examples and to make the record location deterministic for automation, review and audit.

## Contract

```text
Root: reports/daily_observation_records/
Filename: YYYY-MM-DD.json
Example: reports/daily_observation_records/2026-06-01.json
```
