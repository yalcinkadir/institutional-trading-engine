# Alerts

This directory stores machine-readable Entry / Exit Watcher events.

Generated files:

```text
reports/alerts/YYYY-MM-DD-alerts.json
reports/alerts/latest-alerts.json
```

Alert types:

```text
ENTRY_TRIGGERED
STOP_HIT
TARGET_1_HIT
TARGET_2_HIT
EXPIRED
```

These files are generated by:

```bash
python scripts/run_entry_exit_watcher.py
```

The watcher also appends lifecycle events to:

```text
data/signal_lifecycle.jsonl
```

The alert files are intended for:

- intraday review
- Telegram notification integration
- outcome tracking
- future expectancy analysis
- auditability

This is not an execution engine and does not place trades.
